Comprehensive-text is a text generation project based on ChatGPT that aspires to create comprehensive texts incorporating all the most common words in a given language. The goal is to produce linguistically rich, readable content that maximises coverage of a language's core vocabulary — useful for language learning, corpus generation, and NLP benchmarking.
